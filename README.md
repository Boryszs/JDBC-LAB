# JDBC-LAB
## Final task in the subject of Database II

## Run Install 
#### Clone
```sh
git clone https://github.com/Boryszs/JDBC-LAB.git
```
##### Export Database.txt to PostgreSQL
##### Change URL Login Password on Main class

- Swing
- PostgreSQL
